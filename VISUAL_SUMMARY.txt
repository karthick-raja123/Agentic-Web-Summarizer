📊 QUALITY METRICS (6 DIMENSIONS)
════════════════════════════════════════════════════════════════════════════════

  1. RELEVANCE (35% weight)     → How well summary aligns with source
     Range: 0.0 (off-topic) to 1.0 (perfect)
     
  2. COVERAGE (25% weight)      → How comprehensively main points covered
     Range: 0.0 (misses all) to 1.0 (covers all)
     
  3. COHERENCE (20% weight)     → Logical consistency without repetition
     Range: 0.0 (incoherent) to 1.0 (perfect)
     
  4. CONCISENESS (20% weight)   → Appropriate length for content
     Range: 0.0 (too verbose) to 1.0 (appropriate)
     
  5. REDUNDANCY (Bonus)         → Duplicate point detection
     Range: 0.0 (no duplicates) to 1.0 (fully redundant)
     
  6. OVERALL QUALITY (Composite) → Weighted average
     Grade: A+ (90%+), A (80%+), B (70%+), C (60%+), D (50%+), F (<50%)
     Status: PASS (>60%), WARNING (40-60%), FAIL (<40%)
