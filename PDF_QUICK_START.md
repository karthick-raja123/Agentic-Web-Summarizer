### 2. Get a Summary

## Technical Details

### PDF Generation Function
```python
from pdf_generator import generate_summary_pdf

# Simple usage
pdf_bytes = generate_summary_pdf(summary_text)

# With metadata
pdf_bytes = generate_summary_pdf(
    summary=summary_text,
    title="Custom Title",
    url="https://source-url.com"
)
```

### Streamlit Integration
```python
# In the app after summary generation
pdf_bytes = generate_summary_pdf(summary, url=urls[0])

st.download_button(
    label="📕 Download as PDF",
    data=pdf_bytes,
    file_name="summary.pdf",
    mime="application/pdf"
)
```

## Troubleshooting

### PDF button not showing
→ Check if summary generation succeeded
→ Check browser console for errors

### File download fails
→ Check disk space
→ Try different browser
→ Check network connection

### PDF looks strange
→ Original content might have special characters
→ Try viewing in Adobe Reader
→ PDF standards are followed
